# Zevent 🎟️

A web-based event management and ticket booking platform built using **PHP & MySQL**.

## 🌐 Live Website
👉 https://zevent.liveblog365.com

## ✨ Features
- User & Host authentication
- Event creation and approval flow
- Ticket booking with QR verification
- Admin dashboard & analytics
- Fully responsive UI

## 🛠️ Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
